Getting Help

If you need assistance or have any questions about using the Service, here are some ways you can get support:

Frequently Asked Questions (FAQ)

We’ve compiled a list of common questions and answers to help you resolve issues quickly. Check out the FAQ section to find answers to common queries about how the Service works, setting up the JWT token, and troubleshooting tips.

User Guide

For step-by-step instructions on how to use the Service, refer to the User Guide. This guide includes detailed instructions on how to integrate the add-on with Google Sheets and how to use it to manage your App Store Connect translations.

Troubleshooting

If you're encountering an issue, please try the following troubleshooting steps:

  • Ensure that your JWT token is correctly inputted into your Google Sheets.
  • Check for any interruptions in your internet connection.
  • Ensure your Google Sheets account is properly synced with the Service.
  • If the issue persists after trying these steps, please reach out to us.
